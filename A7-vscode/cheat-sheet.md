# cheat sheet

| key                          | value                           |
| ---------------------------- | ------------------------------- |
| Ctrl+I                       | 选中当前行                      |
| Shift+Alt+F                  | 代码格式化                      |
| Shift+Alt+Up或Shift+Alt+Down | 向上/向下复制一行               |
| Ctrl+B                       | 切换侧边栏                      |
| Ctrl+Shift+G                 | 让侧边栏显示Git操作             |
| Ctrl+shift+E                 | 让侧边栏显示资源管理器          |
| Alt+Shift+A                  | 多行注释                        |
| ctrl + k, ctrl + 0           | 折叠单前文件所有代码            |
| ctrl + k, ctrl + j           | 展开单前文件所有代码 |
| ctrl + shift + [ | 折叠一块代码 |
| ctrl + shift + ] | 展开一块代码 |
|                              |                                 |
|                              |                                 |
